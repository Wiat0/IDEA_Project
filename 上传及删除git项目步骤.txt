上传项目至git：
1、选择git bash here；然后在窗口输入 git init 
2、去码云仓库，复制地址，输入git remote add origin + 你刚刚在码云复制的那行路径
3、继续输入 git pull origin master 命令，将码云上的仓库pull到本地文件夹
4、将要上传的文件，添加到刚刚创建的文件夹 
5、 使用git add .  （. 表示所有的）或者 git add + 文件名            // 将文件保存到缓存区
6、使用git commit -m '新添加的文件内容描述
7、使用git push origin master ，将本地仓库推送到远程仓库
8、回到码云，刷新试试，就可以看到新建项目了 

1.查看当前仓库代码状况：git status
2.选择add的文件或者全部更新：git add *
3.commit 到暂存区：git commit -m "注释"
4.pull 当前最新代码：git pull
5.push 到github：git push origin master

删除git上存储仓项目：
1、1、复制仓库地址
2、在本地新建一个空文件夹
3、在空文件夹内，右键选择Git Bash Here
4、弹出GIT Bash框
5、克隆远程仓库 :git clone + 你的仓库地址
6.拉取远程仓库 :cd  + 远程仓库名   //根据自己的远程仓库名输入;git pull origin master
7、查看仓库里的文件:dir 
8、选择想要删除的文件夹进行删除: git rm -r --cached + 想要删除的文件夹  
9、提交删除文件夹说明:git commit -m '删除了trash' 
10、更新GitHub远程仓库:git push -u origin master
